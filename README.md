# **Emoji-Based Music Player** 🎶
A fun Python project that detects your emotion using emojis and plays a matching song in your browser!

**Features**
*	Detects emotion from emoji input
*	Plays a song based on mood
*	Simple and beginner-friendly
*	Uses Python's built-in webbrowser module 

**Technologies Used**
*	Python
*	webbrowser module

**How It Works**
1.	User enters an emoji
2.	Program matches emoji to an emotion
3.	Corresponding song is played in browser

**How to Run**
1.	Install Python (if not installed)
2.	Copy the code into a .py file
3.	Run the file: python emoji_music_player.py
4.	Enter an emoji when prompted
